# STRAT Dashboard V2 - Quick Start Guide

## Getting Started

### Accessing the Dashboard
1. Navigate to `http://localhost:3001/dashboard-v2.html`
2. Login with your credentials
3. You'll land on the Overview section

### Navigation
The sidebar contains all major sections:
- **Overview** - Main dashboard with stats and charts
- **My Wallet** - Wallet management
- **DAO Governance** - Proposals and voting
- **NFT Marketplace** - Browse and mint NFTs
- **Advanced Trading** - Trade with order book
- **Social Feed** - Community posts and interactions
- **DeFi Dashboard** - Liquidity, farming, lending
- **Deploy Contract** - Smart contract deployment
- **Mining** - Start/stop mining operations
- **Staking** - Stake tokens for rewards
- **Portfolio** - Analytics and asset breakdown
- **Analytics** - Network statistics
- **Transaction History** - View all transactions

## Key Features

### Theme Toggle
- Click the moon/sun icon in the header
- Switches between dark and light modes
- Preference saved automatically

### Language Selection
- Use the language dropdown in header
- Supports: English, Spanish, Chinese, Japanese
- Changes apply immediately

### Search
- Use the search bar in header
- Searches across all sections
- Type to find features quickly

### Notifications
- Bell icon shows new notifications
- Click to view notification center
- Toast notifications appear for actions

## Common Actions

### Creating a Governance Proposal
1. Go to DAO Governance
2. Click "Create Proposal"
3. Fill in title, description, category
4. Set voting duration
5. Click "Create Proposal"

### Minting an NFT
1. Go to NFT Marketplace
2. Click "Mint NFT"
3. Enter name, description, category
4. Add image URL or upload file
5. Set royalty % and initial price
6. Click "Mint NFT"

### Placing a Trade
1. Go to Advanced Trading
2. Select Buy or Sell
3. Choose order type (Limit/Market/Stop)
4. Enter price and amount
5. Click "Place Order"

### Staking Tokens
1. Go to Staking
2. Enter amount to stake
3. Select lock period
4. Click "Stake Now"
5. Confirm transaction

### Deploying a Smart Contract
1. Go to Deploy Contract
2. Select contract type or use custom
3. Edit contract code
4. Click "Compile" to check for errors
5. Click "Deploy Contract"

## Tips and Tricks

1. **Quick Navigation**
   - Use sidebar icons to switch sections
   - Click logo to return to overview

2. **Filtering**
   - Use filter buttons in each section
   - Combine multiple filters for precision

3. **Notifications**
   - Green = Success
   - Red = Error
   - Blue = Info
   - Yellow = Warning

4. **Responsive Design**
   - Works on mobile devices
   - Sidebar collapses on small screens
   - Touch-friendly interface

5. **Data Refresh**
   - Most data updates automatically via WebSocket
   - Use refresh buttons for manual updates

## Keyboard Shortcuts

- `Ctrl/Cmd + K` - Focus search
- `Esc` - Close modals
- `Tab` - Navigate form fields

## Troubleshooting

### Charts Not Loading
- Check browser console for errors
- Ensure Chart.js is loaded
- Refresh the page

### WebSocket Not Connecting
- Verify server is running
- Check network connection
- Look for firewall blocks

### Theme Not Saving
- Check browser local storage
- Clear cache and try again

### Language Not Changing
- Ensure translation data is loaded
- Refresh page if needed
