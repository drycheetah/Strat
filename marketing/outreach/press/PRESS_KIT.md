## Elevator Pitch

STRAT is a proof-of-work blockchain built from scratch with JavaScript smart contracts, making blockchain development accessible to 13+ million JavaScript developers worldwide. No forks, no ICO, no pre-mine - just pure innovation.

## The Story

In a world of blockchain forks and clones, STRAT stands out: every line of code written from scratch. The project began with a simple question: "What if blockchain development used the world's most popular programming language?"

Three months of intense development later, STRAT launched as a complete blockchain platform featuring proof-of-work consensus, UTXO transactions, JavaScript smart contracts, and a fully functional P2P network.

## FAQ for Press

**Q: What makes STRAT different?**
A: STRAT is built entirely from scratch with JavaScript smart contracts, making blockchain accessible to millions of existing developers without requiring them to learn new languages.

**Q: Is this another Ethereum fork?**
A: No. Every line of STRAT is original code, not forked from any existing blockchain.

**Q: How does STRAT ensure security?**
A: We use battle-tested cryptography (secp256k1), proof-of-work consensus, and the UTXO model proven by Bitcoin. All code is open source for public audit.

**Q: What's the token distribution?**
A: 100% fair launch via mining. No pre-mine, no ICO, no VC allocation. All STRAT comes from mining rewards.

**Q: Can anyone mine STRAT?**
A: Yes. STRAT is CPU-mineable from any computer. No specialized hardware required.

**Q: How do JavaScript smart contracts work?**
A: Contracts are written in JavaScript, sandboxed for security, and execute with gas metering to prevent abuse.

**Q: Is STRAT production-ready?**
A: Yes. Mainnet is live with real nodes processing real transactions.

**Q: How can developers get started?**
A: Visit our documentation at docs.stratblockchain.io for complete guides and examples.
