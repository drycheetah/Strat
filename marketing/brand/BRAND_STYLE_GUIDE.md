# STRAT Brand Style Guide

## Brand Overview

STRAT is a developer-first blockchain platform built from scratch with proof-of-work consensus, UTXO transactions, and JavaScript smart contracts. Our brand reflects our core values: technical excellence, accessibility, transparency, and innovation.

---

## Brand Values

### 1. **Technical Excellence**
We build on proven technologies and solid engineering principles. Every line of code is intentional.

### 2. **Accessibility**
Blockchain should be accessible to all developers, not just cryptography experts. JavaScript is our bridge.

### 3. **Transparency**
Open source. Open development. Open community. No secrets, no hidden agendas.

### 4. **Innovation**
Building from scratch allows us to innovate without legacy constraints. We question assumptions.

### 5. **Pragmatism**
We choose what works over what's trendy. Real solutions over hype.

## Visual Identity

### Color Palette

**Primary Colors:**

**STRAT Blue**
- Hex: `#0066FF`
- RGB: `0, 102, 255`
- CMYK: `100, 60, 0, 0`
- Usage: Primary brand color, CTAs, headers

**Dark Navy**
- Hex: `#001F3F`
- RGB: `0, 31, 63`
- CMYK: `100, 80, 40, 50`
- Usage: Text, backgrounds, emphasis

**Tech Gray**
- Hex: `#333333`
- RGB: `51, 51, 51`
- CMYK: `0, 0, 0, 80`
- Usage: Body text, secondary elements

**Secondary Colors:**

**Success Green**
- Hex: `#00CC66`
- RGB: `0, 204, 102`
- Usage: Success states, confirmations, positive metrics

**Warning Orange**
- Hex: `#FF9900`
- RGB: `255, 153, 0`
- Usage: Warnings, important notices

**Error Red**
- Hex: `#FF3333`
- RGB: `255, 51, 51`
- Usage: Errors, critical alerts

**Neutral Colors:**

**Light Gray**
- Hex: `#F5F5F5`
- RGB: `245, 245, 245`
- Usage: Backgrounds, subtle borders

**Medium Gray**
- Hex: `#999999`
- RGB: `153, 153, 153`
- Usage: Disabled states, secondary text

**White**
- Hex: `#FFFFFF`
- RGB: `255, 255, 255`
- Usage: Backgrounds, contrast

**Color Usage Guidelines:**
- Use STRAT Blue sparingly for impact
- Dark Navy for professionalism
- Keep 70% neutral, 20% primary, 10% accent
- Ensure WCAG AA contrast ratios (4.5:1 minimum)
- Never use more than 3 colors in a single design

## Typography

### Primary Typeface: Inter

**Headings:**
- Font: Inter Bold
- Sizes: 48px (H1), 36px (H2), 28px (H3), 22px (H4)
- Line Height: 1.2
- Letter Spacing: -0.02em
- Usage: All headings, titles, CTAs

**Body Text:**
- Font: Inter Regular
- Size: 16px (body), 14px (secondary)
- Line Height: 1.6
- Letter Spacing: 0
- Usage: Paragraphs, descriptions

**Code/Technical:**
- Font: Fira Code (monospace)
- Size: 14px
- Line Height: 1.5
- Usage: Code blocks, technical data, addresses

**Alternative:** If Inter unavailable, use:
- Fallback 1: -apple-system, BlinkMacSystemFont
- Fallback 2: "Segoe UI", Roboto
- Fallback 3: Arial, sans-serif

**Typography Guidelines:**
- Never use all caps except for STRAT brand name
- Use bold for emphasis, not italics
- Minimum 16px for body text (accessibility)
- Maximum line length: 75 characters
- Use proper hierarchy (size + weight)

## Voice and Tone

### Writing Style

**General Guidelines:**
- Write in active voice
- Use short sentences and paragraphs
- Explain technical concepts clearly
- Avoid jargon when possible
- Define technical terms when necessary

**Word Choices:**

**Use:**
- Build, create, develop
- Learn, understand, explore
- Open source, transparent
- Decentralized, distributed
- Proof-of-work, consensus

**Avoid:**
- Revolutionary, game-changing (overused)
- Moonshot, lambo (meme culture)
- Guaranteed, risk-free (misleading)
- Disrupting, killing (aggressive)
- Complex jargon without explanation

### Tone by Context

**Technical Documentation:**
- Precise and detailed
- Assumes technical knowledge
- Provides code examples
- Links to resources

**Marketing Content:**
- Approachable and clear
- Highlights benefits
- Demonstrates value
- Calls to action

**Social Media:**
- Friendly and engaging
- Conversational
- Educational
- Community-focused

**Support/Help:**
- Patient and helpful
- Step-by-step instructions
- Empathetic to user frustration
- Solution-oriented

## Content Guidelines

### Blog Posts

**Structure:**
1. **Hook** - Grab attention in first paragraph
2. **Context** - Explain the problem or topic
3. **Deep Dive** - Technical details with examples
4. **Conclusion** - Summary and call to action

**Best Practices:**
- Include code examples
- Use headings for scanability
- Add images/diagrams
- Link to resources
- Proofread carefully

### Social Media

**Twitter/X:**
- 280 characters or thread
- One main idea per tweet
- Use relevant hashtags (2-3 max)
- Include links when relevant
- Engage with replies

**Reddit:**
- Detailed, value-adding posts
- Respond to all comments
- Be helpful, not promotional
- Share code examples
- Link to documentation

**LinkedIn:**
- Professional tone
- Business value focus
- Career/industry insights
- Network with industry leaders
- Share milestones

### Technical Documentation

**Requirements:**
- Clear prerequisites
- Step-by-step instructions
- Code examples with comments
- Expected output shown
- Troubleshooting section

**Best Practices:**
- Test all instructions
- Keep updated with code changes
- Include screenshots
- Provide context
- Link related docs

## Design Patterns

### Website Elements

**Buttons:**
- Primary: STRAT Blue background, white text
- Secondary: Transparent background, blue border
- Disabled: Gray background, gray text
- Padding: 12px 24px
- Border radius: 4px
- Font: Inter Semi-Bold

**Cards:**
- Background: White
- Border: 1px solid #E5E5E5
- Border radius: 8px
- Shadow: 0 2px 8px rgba(0,0,0,0.1)
- Padding: 24px

**Forms:**
- Label: Inter Medium, 14px, above input
- Input: 16px text, 12px padding
- Border: 1px solid #CCCCCC
- Focus: 2px STRAT Blue border
- Error: Red border + error message below

**Navigation:**
- Top bar: Dark Navy background
- Links: White text, hover underline
- Mobile: Hamburger menu
- Logo: Top left
- CTA: Top right

### Code Blocks

**Inline Code:**
- Background: #F5F5F5
- Border: 1px solid #E0E0E0
- Font: Fira Code, 14px
- Padding: 2px 6px
- Border radius: 3px

**Block Code:**
- Background: #282C34 (dark theme)
- Font: Fira Code, 14px
- Line numbers: Optional
- Syntax highlighting: Yes
- Copy button: Top right
- Padding: 16px
